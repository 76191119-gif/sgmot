🚀 CÓMO DESPLEGAR A PRODUCCIÓN
════════════════════════════════════════════════════════════════════════════════

PASO 1: Ejecutar deployment script (valida todo automáticamente)
────────────────────────────────────────────────────────────────
Windows (PowerShell como Admin):
  > cd C:\xampp\htdocs\sgmot
  > .\deploy.ps1

Linux/Mac:
  $ cd /var/www/sgmot
  $ bash deploy.sh

PASO 2: Configurar credenciales CRÍTICAS
────────────────────────────────────────────────────────────────
1. JWT_SECRET - Generar aleatorio:
   $ openssl rand -base64 32
   → Guardar en api/.env

2. CORS_ALLOWED_ORIGIN - Cambiar a dominio real:
   CORS_ALLOWED_ORIGIN=https://tudominio.com

3. DB_PASSWORD - Crear usuario MySQL seguro:
   $ mysql -u root -p
   > CREATE USER 'sgmot_user'@'localhost' IDENTIFIED BY 'CONTRASEÑA_FUERTE';
   > GRANT ALL ON sgmot.* TO 'sgmot_user'@'localhost';
   → Guardar en api/.env

PASO 3: Verificar seguridad
────────────────────────────────────────────────────────────────
$ php api/security-check.php
→ Debería mostrar: ✅ SECURITY CHECK PASSED

PASO 4: Configurar servidor web
────────────────────────────────────────────────────────────────
Apache:
  - mod_rewrite habilitado
  - mod_headers habilitado
  - mod_deflate habilitado
  - HTTPS con certificado SSL válido

Nginx:
  - Config en DEPLOYMENT.md

PASO 5: Importar base de datos
────────────────────────────────────────────────────────────────
$ mysql -u sgmot_user -p sgmot < database/sgmot.sql

PASO 6: Configurar logs y monitoreo
────────────────────────────────────────────────────────────────
$ mkdir -p logs
$ chmod 777 logs
→ Ver instrucciones en DEPLOYMENT.md
